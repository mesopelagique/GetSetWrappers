<!-- 4D date wrapper this utility getters and setters -->
# Date  

## init

```4d
$date:=cs.Date.new()
$date:=cs.Date.new(Current date)
$date:=cs.Date.new(Date("25/12/97"))
```

## getters

```4d
$value:=$date.year
$value:=$date.month
$value:=$date.day
```

## setters

```4d
$date.year:=2021
$date.month:=04
$date.day:=18
```
